Add-on to WPML and to WPMU Sitewide tags plugins. Will propagate WPML language properties to the Network wide aggregation blog. 

== Description ==

This plugin is an add-on to the [WPML Multilingual CMS plugin](http://wpml.org/) from http://wpml.org.

It is also an add-on to the [WordPress MU Sitewide Tags Pages plugin from Donncha O Coimh](http://ocaoimh.ie/wordpress-mu-sitewide-tags/ "WordPress MU Sitewide Tags Pages plugin from Donncha O Coimh") (aka. SWT).

It will make it possible to have site-wide contents, search, tags and categories in multiple languages.

= Network-wide multilingual support =

This WordPress 3.0 multisite or WPMU plugin makes sure that all data replicated to a central aggregation blog by the WordPress MU Sitewide Tags Pages plugin keeps its language properties generated by the WPML Multilingual CMS plugin.

Base package includes .pot file for translation of the interface, and English and French versions.

= Installing WPML Network-wide =

You will need my other plugin [YD Network-wide Options](http://wordpress.org/extend/plugins/yd-wpmu-sitewide-options/) to automatically install and setup WPML on all sub-blogs and replicate settings changes on all blogs.

== Installation ==

1. Unzip yd-network-wide-wpml.zip
1. Upload the `yd-network-wide-wpml` directory and all its contents into the `/wp-content/plugins/` directory of your main site
1. Activate the plugin **on your main site** or **network-wide** through the 'Plugins' menu in WordPress

For specific installations, some more information might be found on the [YD Network Wide WPML plugin support page](http://www.yann.com/en/wp-plugins/yd-network-wide-wpml "Yann Dubois' Network-wide WPML plugin for multisite WordPress")
